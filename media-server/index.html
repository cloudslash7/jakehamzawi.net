HOME MEDIA SERVER

Who needs Netflix? Setting up a home media server is easier than you think!

Requirements

Plex Account

It's free to make an account and use Plex. Upgrading to Plex Pass gives you some additional functionality, but it's generally not necessary.

Computer

Almost any computer will do - an old desktop, a raspberry pi, basically anything! The less power it uses, the better.

Handbrake

This is really good software for ripping video files from dvd's and bluray's.

Mass Storage

Depending on how much media you have, you'll need a certain size of hard drive. I'd say a decent sized library of 100 movies and 20ish shows would be 2TB. It's better to overestimate!

Good Upload Speed

For remote access (when you're away from home or if others use your server), you'll need an upload speed of around 10Mbps or more.
